# scripts-vault

Repositorio personal para guardar scripts utiles y bien organizados por lenguaje.

## Estructura

- `python_scripts/`: scripts escritos en Python

Con el tiempo se pueden agregar carpetas como `javascript_scripts/`, `java_scripts/` u otras segun el lenguaje.

## Como usar este repositorio

- Cada carpeta de lenguaje contiene los scripts de ese lenguaje.
- Dentro de cada carpeta tambien hay un archivo con instrucciones para instalar o preparar ese entorno en tu computadora.
- Cada script puede incluir comentarios propios sobre las librerias que necesita o como ejecutarlo.

## Ejemplo actual

- En [python_scripts/remove_bg_to_webp.py](C:/Dev/scripts-vault/python_scripts/remove_bg_to_webp.py) esta el primer script de ejemplo.
- En [python_scripts/PYTHON_SETUP.md](C:/Dev/scripts-vault/python_scripts/PYTHON_SETUP.md) estan las instrucciones para instalar Python en Windows y preparar esa carpeta.

## Idea del proyecto

Mantener un vault de scripts buenos, ordenados por lenguaje, con instrucciones claras por carpeta para que cualquier persona pueda preparar el entorno y ejecutarlos.
